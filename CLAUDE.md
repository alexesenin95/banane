# CLAUDE.md — контекст проекта для Claude Code

## Что это
2D-платформер «Банановый поход» на **Phaser 3** (CDN в index.html). Герой-обезьяна против орков и босса-паука.
Перенесён из чат-прототипа Claude для нормальной разработки.

## Запуск
Нужен локальный веб-сервер (ассеты грузятся из файлов): `npx serve`, затем открыть localhost-адрес.
`play-offline.html` — автономная сборка для быстрой проверки.

## Структура
- index.html — разметка (оверлеи: меню/интро/кат-сцена/инвентарь/конец) + Phaser + js/game.js.
- css/style.css — UI; фон меню через --menuBg.
- js/game.js — ВСЯ логика одним файлом. Секции: история/интро, инвентарь, кат-сцены, genLevel, create/update, spawnEnemy + ИИ, бой (doAttack/damageEnemy/hitEnemy), помощники.
- assets/sprites — герой, враги, снаряды, барьеры, плитка, иконки оружия. assets/bg — параллакс ×3. assets/portraits — Чичо/Доня/орк/герой. assets/ui — фон меню.
- TEX в game.js — манифест ключ→путь, грузится в preload.

## Системы (работают)
Уровни: 3, процедурная генерация (~20000px, пропасти/платформы/башни). Враги: креветка, орк, берсерк, ножеметатель, лучник (снаряды eprojs), щитоносец (периодическая неуязвимость inv), босс-паук. Бой: стомп + оружие (орк = 4 стомпа ИЛИ 6 ударов). Инвентарь (I): 2 активных + 8 заготовок (WEAPON_LIST). Прокачка на 400 очков. Броня (B): −500 очков → 5 сек неуязвимости. Кат-сцены вызывают спавн орков (cuts[].spawn). Барьеры: шипы/колючки (урон), камень (ломается оружием), частокол (твёрдый), ворота (декор). Чекпоинт cpX. ADMIN-флаг: клавиши [ ] (перед релизом ADMIN=false).
