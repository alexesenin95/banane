# Банановый поход

2D-платформер на Phaser 3 (обезьяна-воин против орков в Долине животных).

## Что внутри
```
index.html           — страница игры (грузит ассеты из /assets)
css/style.css         — стили меню/интерфейса
js/game.js            — вся игровая логика
assets/               — картинки отдельными файлами (sprites/ bg/ portraits/ ui/)
play-offline.html     — та же игра ОДНИМ файлом (играть без настройки)
CLAUDE.md             — описание проекта для Claude Code
```

## Просто поиграть прямо сейчас
Открой **play-offline.html** двойным кликом — запустится в браузере без настройки.

## Шаг 1. Залить на GitHub (через сайт)
1. Зайди в свой репозиторий на github.com.
2. **Add file → Upload files**.
3. Перетащи папки `css`, `js`, `assets` и файлы `index.html`, `README.md`, `CLAUDE.md` (и при желании `play-offline.html`). Структура папок сохранится.
4. **Commit changes**.

## Шаг 2. Открыть в Claude Code (Windows)
1. Установи **Node.js** (LTS): https://nodejs.org
2. Установи Claude Code — оф. инструкция: https://docs.claude.com/en/docs/claude-code/overview
   Обычно: `npm install -g @anthropic-ai/claude-code` (если не сработает — следуй шагам со страницы выше).
3. Открой папку проекта (или подключи свой GitHub-репозиторий) в Claude Code и пиши, что делаем дальше.

## Шаг 3. Запустить для разработки
НЕ открывай `index.html` двойным кликом (браузер не загрузит картинки из файлов). Нужен локальный сервер:
- В Claude Code: «запусти локальный сервер для игры».
- Или вручную в папке проекта: `npx serve` → открой показанный http://localhost-адрес.
(`play-offline.html` работает и без сервера.)

## Управление
Стрелки — движение, Пробел/↑ — прыжок, X/Z/F или клик — атака, 1/2 — смена оружия, I — инвентарь, B — броня за 500 бананов. `[` `]` — смена уровней (админ; флаг `ADMIN` в js/game.js).
